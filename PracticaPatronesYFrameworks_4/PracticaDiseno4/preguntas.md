## 4. a) Escenario de modificabilidad

**Problema:** Escribe un escenario de modificabilidad que refleje lo que se pregunta en el punto.

| Elemento                | Descripción                                             |
| ----------------------- | ------------------------------------------------------- |
| **Fuente del estímulo** | Desarrollador                                           |
| **Estímulo**            | Cambiar el numero de ventanas                                    |
| **Artefacto**           | application.properties                                                  |
| **Entorno**             | Entorno de desarrollo                                   |
| **Respuesta**           | El programa se ejecuta correctamente al cambiar el numero de ventanas en el archivo application.properties                    |
| **Respuesta mediada**   | El cambio se realiza en menos de 1 minuto y el programa se ejecuta correctamente sin afectar otros archivos
